# Calm Sky Explorer

Find the quiet answer to where that plane is headed.

A minimalist real-time aircraft tracking app that displays nearby flights on an interactive map.

<img width="831" height="744" alt="image" src="https://github.com/user-attachments/assets/d1d0ce5b-1e1b-44e9-87ec-7f448674b91d" />


## Features

- **Real-time aircraft tracking** - See flights in your vicinity
- **Interactive map** - Zoom and pan to explore any area
- **Return to my location** - Center the map on your current position
- **Minimalist design** - Clean interface focused on what matters

## Tech Stack

- Next.js 16
- React
- Leaflet (OpenStreetMap)
- Vercel

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Deployment

Deployed on Vercel: https://calm-sky-explorer.vercel.app/
